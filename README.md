# 🐍 Cobrinha

Jogo da cobrinha clássico rodando direto no terminal do Windows, com visual estilo **cmd**.

Feito em **Python puro** — sem nenhuma dependência externa.

## Como jogar

Requisito: [Python 3](https://www.python.org/downloads/) instalado.

```
python cobrinha.py
```

Ou simplesmente dê dois cliques em **`jogar.bat`** — ele abre uma janela do cmd com o jogo.

## Controles

| Tecla | Ação |
|---|---|
| Setas ou `W` `A` `S` `D` | Mover a cobrinha |
| `R` | Reiniciar a partida |
| `Q` ou `ESC` | Sair |

## Regras

- Coma a comida (`*`) para crescer e marcar pontos.
- A cada comida, o jogo fica um pouco mais rápido.
- Bater na parede ou no próprio corpo encerra a partida.
- O recorde da sessão fica no placar.

## Dica

Use a janela do terminal com pelo menos **42 colunas × 24 linhas** para o tabuleiro caber sem quebrar.
